# project
Создание сайта для ослеживания госзакупок.
1. Задача проекта
- Создать персональный сайт для автоматизированного мониторинга, поиска и управления участием в государственных закупках по 44-ФЗ и 223-ФЗ.
- Основная цель — сэкономить мое время,  на ручном поиске тендеров в ЕИС и на ЭТП - таких ка - РосЭлТорг, Сбербанк-АСТ, и т.д
-  Сайт поможет находить релевантные лоты по ключевым словам, сохранять их для анализа и готовить базовые документы для подачи.
-  В итоге: повысить шансы на выигрыш тендеров без перегрузки рутиной.
2. Функционал сайта
2.1.Сайт должен быть простым в использовании.
- Должнен быть поиск по ключевым словам.
  Основная страница — поисковая форма. Вводишь слова (напр."лабораторные мыши"  + фильтры: регион, цена, сроки).
  Сайт интегрируется с ЕИС (zakupki.gov.ru) или парсит публичные данные.
  Результат: список тендеров с краткой инфо (номер, заказчик, НМЦК,сайт откуда взят).  
  Как использовать: Ежедневно проверяешь "горячие" тендеры по подписке — уведомления на email/SMS.
- Так же сайт должен быть с логином и паролем.

2.2.Сохранение и управление тендерами:  
  - Кнопка "Сохранить" для каждого лота — добавляется в личный список "Избранное".
  - В нём: просмотр деталей (ТЗ, документы), пометки ( "Подходит/Не подходит"). Экспорт в PDF/Excel для анализа.
    
 2.3. Дополнительные функции:  
  - Автоматический анализ: Краткий обзор (например, "Шанс выигрыша: высокий, если цена < НМЦК на 15%").
  - Подготовка заявки: Генератор форм — заполняешь данные, сайт формирует пакет для ЭТП (с ЭЦП-интеграцией, если расширить).  
   - Уведомления: Push-уведомления о новых лотах или изменениях

3. Задачи:
3.1. Где искать тендеры и данные:  
  - Основной источник: ЕИС (zakupki.gov.ru) — бесплатно, API для парсинга.  
  - ЭТП: РосЭлТорг (roseltorg.ru), Сбербанк-АСТ (sberbank-ast.ru), РТС-тендер (rts-tender.ru)

3.2 На что ориентироваться:  
  - Ищи по ОКПД2-кодам (классификатор товаров/услуг), региону и типу (аукцион/запрос котировок).
  - Задача: Создай фильтры в сайте по твоим нишам (напр."лабораторные животные").

3.3.Где сохранять данные:  
  - Внутренне: База данных сайта (SQLite)   
  - Excel/Google
